# DBAP 617 Dynamic Presentation

This is a self-contained presentation project built from the paper text on the liminal zone in VA mental health work.

## Open in Visual Studio Code
1. Open this folder in VS Code.
2. Install the **Live Server** extension (optional but easiest).
3. Right-click `index.html` and choose **Open with Live Server**.
4. Present in browser fullscreen.

## Controls
- `Right Arrow` or `Space`: next slide
- `Left Arrow`: previous slide
- `F`: fullscreen
- `P`: overview mode
- Click anywhere: next slide

## Files
- `index.html` - slide structure
- `styles.css` - visual design and motion
- `script.js` - slide navigation and dynamic behavior
- `assets/conceptmap.png` - conceptual model image

You can edit the slide text directly in `index.html`.
